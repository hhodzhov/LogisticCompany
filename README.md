<h1>Starting the project</h1>
<p>Start the docker with <code>docker-compose up</code> in order to crate the database.  
</p>
<p>Download and install <a href="https://nodejs.org/en/download/">Node.js</a> and <a href="https://nodejs.org/en/download/">npm</a></p>
<p>Navigate to <code>ng-logistics</code> and type <code>npm install</code>. Once it is compiled start the angular server with <code>npm start</code></p>
<p>Launch you browser with disabled web security flag. The command for Chrome is <code>google-chrome --user-data-dir="/home/(root)/chrome-temp" --disable-web-security</code></p>
<p>Start the java application and go to the <code>localhost:4200</code></p>
